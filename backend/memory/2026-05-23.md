## Work Assistant Core — Visión completa

Said me explicó el objetivo final:

**Qué quiere lograr:**
- Un asistente que automatice sus labores diarias de trabajo
- Monitoreo de actividad REAL del computador (no simulación)
- Integración con ClickUp (donde maneja tareas del día a día)
- Al final del día: reporte automático de qué hizo, en qué tareas trabajó y proyectos
- Eventualmente: un "cerebro" con IA para análisis y automatización

**Stack actual (v0.3):**
- Python backend modular
- Simulación de actividad (temporal)
- EventBus, System State, SQLite
- REST API + WebSocket streaming
- Cross-platform (Win/Mac/Linux)

**Próximos pasos identificados:**
1. Monitoreo real de actividad del SO (ventanas activas, apps)
2. Integración con API de ClickUp
3. Generación de reporte de fin de día
4. Capa de IA para entender contexto y patrones
